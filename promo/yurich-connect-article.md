# Yurich Connect: мой Android-клиент для стабильных VPN-профилей

Я сделал Yurich Connect как удобное Android-приложение для подключения VPN-профилей без лишней возни с конфигами. Главная идея простая: пользователь вставляет подписку, QR-код или отдельную ссылку, а приложение само разбирает профиль, показывает понятный список серверов и помогает держать соединение стабильным в Wi-Fi и мобильных сетях.

Особый упор я сделал на NaiveProxy. Это важный протокол для тех, кому нужен аккуратный HTTPS-профиль, понятная настройка через `naive+https://...` и стабильная работа через sing-box. В Yurich Connect Naive-профили импортируются напрямую, отображаются отдельно в табах, проверяются пингом и работают через защищённый DNS внутри туннеля.

## Какие профили поддерживаются

Yurich Connect поддерживает несколько типов подключений:

- NaiveProxy: основной акцент приложения, поддержка `naive+https://` и sing-box JSON.
- VLESS Reality: быстрый и стабильный профиль для серверов с Reality.
- VLESS TLS: классический VLESS через TLS.
- Hysteria2 и Hysteria: поддержка UDP/QUIC-профилей для будущих серверов.
- Remnawave и raw-подписки: приложение умеет импортировать списки профилей по HTTPS-ссылке.
- sing-box JSON: для ручных и продвинутых конфигураций.

## Почему NaiveProxy здесь важен

Во многих приложениях NaiveProxy либо требует отдельный плагин, либо подключается не всегда предсказуемо. В Yurich Connect я сделал Naive-профили частью основного сценария:

- импорт `naive+https://` из буфера, QR-кода или подписки;
- отдельная вкладка Naive, чтобы не теряться среди VLESS и Hysteria;
- ping каждого профиля прямо в списке;
- DNS через выбранный профиль, чтобы уменьшить DNS-рассинхрон;
- строгий TUN-режим для Wi-Fi и LTE;
- фоновый keeper, который проверяет туннель без открытия приложения;
- диагностика с логами, где пароли и ключи скрываются перед отправкой разработчику.

Пользователь получает не просто “ещё один VPN-клиент”, а приложение, которое заточено под реальные подписки и реальные мобильные сети.

## Чем отличается от других приложений

Yurich Connect не пытается быть перегруженным комбайном на сотни настроек. Его задача — дать пользователю понятный рабочий интерфейс:

- вставил подписку;
- увидел профили по протоколам;
- выбрал сервер;
- проверил пинг;
- подключился;
- при проблеме отправил диагностику.

В приложении есть табы профилей, флаги стран, пинг, срок подписки, уведомление о продлении, автообновления через GitHub Releases, русский и английский интерфейс, QR/буфер/ручной ввод и отдельная вкладка связи с поддержкой.

## Что получает пользователь после установки

После установки Yurich Connect пользователь получает:

- удобный импорт подписки и отдельных VPN-ссылок;
- понятный список профилей с вкладками по протоколам;
- быстрый доступ к NaiveProxy-профилям;
- поддержку VLESS Reality, VLESS TLS, Hysteria2 и sing-box JSON;
- защищённый DNS через туннель;
- фоновое восстановление соединения при сбоях;
- уведомления о новой версии приложения;
- напоминания о сроке платной подписки;
- диагностику для поддержки без раскрытия паролей и UUID.

## Важно про видимость VPN

Yurich Connect защищает IP и DNS-запросы через VPN-туннель. Но Android использует системный `VpnService`, и некоторые приложения могут видеть сам факт включённого VPN. Это ограничение Android, а не отдельного клиента. Моя задача — сделать подключение стабильным, понятным и безопасным для пользователя, без обещаний, которые технически нельзя гарантировать без root или кастомной прошивки.

## Для кого это приложение

Yurich Connect подойдёт тем, кто пользуется подписками, хочет быстро переключаться между серверами и ценит простой интерфейс. Особенно приложение полезно для пользователей NaiveProxy, потому что этот протокол здесь не спрятан в экспериментальные настройки, а вынесен в нормальный рабочий сценарий.
