# Yurich Connect Android v1.0.41

## Что исправлено

- Исправлено обновление подписок формата `https://.../s/.../links.txt`.
- Приложение теперь запоминает исходную ссылку подписки отдельно от ссылок
  профилей. QR, копирование и ручной ввод продолжают работать с протокольной
  ссылкой профиля, а кнопка обновления подписки использует именно `links.txt`.
- Для старых профилей без сохранённой ссылки подписки кнопка обновления больше
  не выглядит сломанной: приложение откроет окно импорта и попросит вставить
  ссылку подписки один раз.
