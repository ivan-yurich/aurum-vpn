# Aurum VPN Android 1.0.20

Дата: 2026-05-25

## Что изменилось

- Добавлена вкладка «Обновления»: приложение проверяет свежий релиз, скачивает
  подходящий APK и открывает установщик Android без перехода на GitHub.
- TUN переведён на `mixed` stack: TCP идёт через системный стек, UDP остаётся
  через gVisor. Это должно помочь NaiveProxy и VLESS на тестах скорости.
- Убран `endpoint_independent_nat`, потому что для текущего режима он не нужен
  и может слегка снижать производительность.

## Что тестировать

1. NaiveProxy на Wi-Fi: скорость и стабильность.
2. NaiveProxy на LTE.
3. VLESS Reality после переключения с NaiveProxy.
4. Вкладка «Обновления»: проверка, скачивание, открытие установщика.
