# Yurich Connect Android v1.0.36

Дата: 2026-06-08

## Что изменилось

- Автообновления Android теперь используют новый репозиторий GitHub:
  `ivan-yurich/Yurich-Connect-Android`.
- Документация обновлена под новый адрес проекта.
- Версия приложения поднята до `1.0.36`, чтобы установленный клиент увидел
  миграционное обновление через встроенную кнопку обновления.
- Вкладка "Профиль и сеть" переделана в компактные строки: протокол, сеть,
  DNS, фоновая стабильность, страна, пинг и сервер.
- DNS теперь подписан как "через профиль": приложение использует строгий TUN,
  FakeIP и выбранный профиль, а не навязывает отдельный публичный DoH.
- Добавлена честная подсказка про Android VPN detection: системные приложения
  могут видеть сам `VpnService`, но Yurich Connect снижает DNS/IP-рассинхрон и
  держит DNS через выбранный профиль.
- Текущая рабочая настройка протоколов для мобильных сетей сохранена; фоновый
  keeper продолжает восстанавливать зависший туннель без открытия приложения.
- Keeper стал быстрее реагировать на зависание туннеля после переключений
  Wi-Fi/LTE: уменьшены стартовая пауза, интервал проверки, cooldown и порог
  ошибок.
