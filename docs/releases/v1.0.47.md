# Yurich Connect Android v1.0.47

## What changed

- Added the new Yurich Connect visual polish layer: app-wide blue gradients,
  glowing connection panel, pulsing uptime button, and highlighted active
  profiles.
- Improved the profile and network cards with a richer glass-like blue surface
  while keeping compact 8 px cards and stable text layout.
- Kept VPN protocol behavior untouched in this release so the working tunnel
  configuration remains stable.
