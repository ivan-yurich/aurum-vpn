# Yurich Connect Android v1.0.40

## Что исправлено

- Исправлена нестабильная проверка обновлений, когда GitHub временно отдаёт
  `HTTP 504`.
- Проверка свежей версии теперь делает несколько попыток с паузами.
- Если GitHub JSON API недоступен, приложение берёт свежий тег через обычную
  страницу `/releases/latest`.
- Загрузка APK теперь тоже повторяется и имеет запасные прямые ссылки на
  `YurichConnect-android-release.apk`.

## Зачем это нужно

На некоторых мобильных сетях и через VPN GitHub иногда отдаёт временный
gateway timeout. Теперь это не должно сразу ломать автообновление.
