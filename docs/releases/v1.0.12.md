# Aurum VPN Android 1.0.12

Дата: 2026-05-23

## Что изменилось

- Улучшена проверка NaiveProxy при старте.
- Если сервер прокидывает IP-адреса, но зависает на доменных CONNECT-запросах,
  приложение больше не отвергает профиль слишком рано.
- Это помогает профилям с рабочим TUN-режимом пережить временно сломанный DNS
  на VPS, но сам DNS сервера всё равно нужно починить.

## Диагностика go-it.tech

- HTTPS сайт на `go-it.tech:443` доступен.
- HTTPS proxy через `go-it.tech` с доменными целями уходит в timeout.
- HTTPS proxy через `go-it.tech` с IP-целью проходит.
- Рабочий `n8n-cloud.online` проходит и доменные, и IP-цели.

Вывод: у `go-it.tech` вероятна проблема серверного DNS/резолвера для Caddy
forwardproxy, а не ошибка формата ссылки в приложении.
