# Aurum VPN Android v1.0.25

## Changes

- Improved long-running VPN stability by throttling traffic counter UI updates
  instead of rebuilding the screen for every native event.
- Added timeouts around native status/start/stop/config calls, with safe
  logging when Android or the plugin does not answer quickly.
- Cleaned up service shutdown paths so receiver, notification, command server,
  network monitor, and TUN descriptor are closed even when Android destroys the
  service.
- Prevented duplicate libbox initialization after Flutter engine reattach.
- Hardened release data handling by disabling Android backup/data extraction
  and cleartext app traffic.

## Validation

- `dart format`
- `flutter analyze`
- `flutter test`
- `flutter build apk --release --split-per-abi`
- `apksigner verify`
