# Aurum VPN Android 1.0.14

Дата: 2026-05-25

## Что изменено

- NaiveProxy больше не стартует через экспериментальный QUIC/H3 по умолчанию.
- Автоподключение NaiveProxy снова использует стабильный порядок:
  native Naive HTTP/2, затем HTTPS CONNECT fallback.
- Сохранён IPv4-only dial до сервера, чтобы не ловить битые IPv6-маршруты.
- TCP Fast Open отключён в дефолтном outbound, чтобы не провоцировать
  переподключения на мобильных сетях и домашних роутерах.
- Окно добавления профиля открывается по центру экрана.
- APK собраны раздельно по ABI, чтобы не ставить универсальный файл почти
  200 МБ.
