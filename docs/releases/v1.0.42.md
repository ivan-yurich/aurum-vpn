# Yurich Connect Android v1.0.42

## Что исправлено

- Срок подписки теперь подтягивается из названий профилей, если сервер не отдаёт
  отдельный `subscription-userinfo` заголовок.
- Поддержан формат из твоей подписки: `🇫🇮 Finland • до 08.06.2027 • Yurich
  Proxy`.
- Также поддержаны варианты `until 08-06-2027`, `expire`, `expires` и `expiry`.
- Явные данные от сервера всё равно остаются главными, если они есть.

## Как проверить

Обнови приложение, нажми обновление подписок и выбери профиль из ссылки
`links.txt`. В блоке "Профиль и сеть" вместо "Срок не указан" должен появиться
остаток срока.

## Проверки

- `flutter analyze`
- `flutter test`
- `flutter build apk --release`
- `apksigner verify --verbose`
