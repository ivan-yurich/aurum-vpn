# Yurich Connect Android v1.0.45

## Что изменилось

- Нижняя часть приложения стала аккуратнее: связь, язык, обновления, FAQ и логи
  теперь собраны в один центр приложения.
- Добавлена тихая проверка новых версий при запуске.
- Если вышла свежая версия, приложение подсвечивает раздел обновлений, показывает
  сообщение внутри приложения и пробует отправить системное уведомление Android.
- Установка обновления остаётся по кнопке, чтобы Android-разрешения и установщик
  работали предсказуемо.

## Проверки

- `flutter analyze`
- `flutter test`
- `flutter build apk --release`
- `apksigner verify`
