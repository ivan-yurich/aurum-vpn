# Yurich Connect Android v1.0.43

## Что исправлено

- Усилено определение срока подписки для списков профилей.
- Если выбранный профиль называется просто `net-it.pro`, но другой профиль в
  этой же подписке содержит дату вроде `until-2027-06-08` или `до 08.06.2027`,
  приложение теперь применит этот срок ко всей подписке.
- Добавлена поддержка формата даты `until-2027-06-08`.

## Как проверить

1. Обнови приложение.
2. Нажми обновление подписок.
3. Выбери профиль `net-it.pro`.
4. В блоке "Профиль и сеть" строка "Подписка" должна показать срок вместо
   "Срок не указан".

## Проверки

- `flutter analyze`
- `flutter test`
- `flutter build apk --release`
- `apksigner verify --verbose`
