# Aurum VPN Android 1.0.21

Дата: 2026-05-26

## Что изменилось

- Для NaiveProxy теперь первым пробуется HTTPS CONNECT fallback, а native Naive
  остаётся вторым вариантом. Это убирает лишний native Naive UDP/QUIC путь на
  старте и может дать стабильнее скорость на Android.
- TUN возвращён на `gvisor`, потому что актуальная документация NB4A отмечает,
  что новые версии снова используют gVisor по умолчанию для совместимости.
- MTU 9000, FakeDNS, DNS через туннель, VLESS и Hysteria/Hysteria2 оставлены.

## Важно

Предупреждение Яндекс Интернетометра о VPN обычно зависит от репутации
выходного IP/ASN. Для `n8n-cloud.online` IP находится в датацентровом ASN, и
это нельзя надёжно скрыть настройкой Android-клиента без смены выходного IP.

## Что тестировать

1. NaiveProxy на Wi-Fi: скорость и предупреждение Яндекса.
2. NaiveProxy на LTE.
3. Переключение NaiveProxy -> VLESS Reality -> NaiveProxy.
4. Диагностика должна показывать `mode=https-connect` для первого успешного
   Naive-подключения.
