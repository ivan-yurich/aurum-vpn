# Aurum VPN Android 1.0.11

Дата: 2026-05-23

## Что изменилось

- Добавлен fallback для NaiveProxy: приложение пробует native Naive и HTTPS CONNECT режимы.
- Перед принятием Naive-подключения выполняется проверка интернета через локальный mixed-proxy.
- Добавлен импорт одиночного sing-box outbound JSON с `type: http` и `type: naive`.
- Улучшены ожидание остановки VPN, переключение профилей и watchdog статуса.
- Убран технический текст с упоминанием сторонних клиентов.

## Проверки

- `flutter test`
- `flutter build apk --release`
