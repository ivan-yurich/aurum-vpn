# Aurum VPN Android v1.0.10

Release date: 2026-05-23

## Fix

- `naive+https://` links now build a native sing-box `naive` outbound on
  Android.
- The previous Android config forced NaiveProxy links into generic HTTP CONNECT,
  which can fail on NaiveProxy servers that expect the native Naive outbound.

- `flutter analyze`
- `flutter test`
- `flutter build apk --release`
