# Yurich Connect Android v1.0.46

## Что изменилось

- Усилен фоновый Android keeper для ночного режима и долгого простоя телефона.
- Возвращены разрешения `WAKE_LOCK` и восстановление после загрузки устройства.
- Если VPN был включён пользователем и конфиг сохранён, приложение может поднять
  VPN после ребута, обновления APK или неожиданного уничтожения сервиса.
- Native watchdog теперь проверяет туннель в фоне и перезапускает sing-box после
  нескольких реальных провалов, а не по одному случайному сбою.
- Ручное отключение не запускает VPN обратно: восстановление работает только для
  неожиданных остановок.
