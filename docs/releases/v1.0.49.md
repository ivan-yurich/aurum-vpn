# Yurich Connect Android v1.0.49

## What changed

- Pinned the connection panel above the profile scroller so the connect button,
  uptime, and live traffic stay on screen with large subscriptions.
- Added a clear active badge to the profile that is currently driving the VPN
  tunnel.
- Strengthened selected profile and profile-tab highlighting so the chosen
  location/protocol remains visible even while VPN is stopped.
