# Yurich Connect Android v1.0.48

## What changed

- Added VLESS XHTTP and VLESS mKCP recognition in imports, profile tabs, and
  labels.
- Kept the Android tunnel safe by blocking unsupported XHTTP/mKCP connection
  attempts with a clear Xray/libXray requirement. The current Android core is
  sing-box based, and it should not emit invalid configs for transports it does
  not support.
- Added regression coverage for subscription URLs like `/s/<token>/`, including
  sources without a visible `links.txt` suffix.
- Updated refresh copy so saved subscriptions can use either `/s/.../` or
  `/links.txt`.
