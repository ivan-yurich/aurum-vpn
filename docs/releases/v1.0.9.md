# Aurum VPN Android v1.0.9

Release date: 2026-05-23

This is the Android-only release for the `ivan-yurich/aurum-vpn` repository.
Windows builds are maintained separately in
[`ivan-yurich/aurum-vpn-windows`](https://github.com/ivan-yurich/aurum-vpn-windows).

## Highlights

- Removed extra top action icons from the Android UI.
- Fixed the connection status card height so traffic counters do not stretch the
  layout.
- Moved language switching to the contact section.
- Rewrote user-facing helper text.
- Kept the current Android protocol tuning: protected DNS through the VPN,
  Wi-Fi/LTE TUN settings, gVisor stack, strict routing, NaiveProxy HTTPS
  CONNECT compatibility, and explicit stop/start profile switching.
