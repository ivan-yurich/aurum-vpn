# Aurum VPN v1.0.9

Release date: 2026-05-23

## Summary

This release prepares Aurum VPN for public GitHub distribution with Android and
Windows artifacts. The Android UI was simplified, the connection card was fixed
to avoid layout jumps, and the profile/network text was rewritten for users.

## What's New

- Android top bar now shows only the Aurum VPN icon and title.
- Connection card uses fixed-height traffic counters so speed changes do not
  stretch the UI.
- Language switching moved to the contact section.
- Profile/network helper text is shorter and easier to understand.
- Contact and project links are prepared for GitHub:
  - Site: https://ivan-it.net
  - Email: ai@ivan-it.net
  - VK: https://vk.com/ivan_yurievich_it
  - Donate: https://dzen.ru/ivanyurievich?donate=true
