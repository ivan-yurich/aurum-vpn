# Yurich Connect Android v1.0.38

Дата: 2026-06-08

## Что изменилось

- Добавлено определение страны профиля и кеширование результата в профиле.
- Карточки профилей показывают флаг из имени подписки, по геолокации endpoint
  IP или по реальной стране выхода после подключения через локальный туннель.
- Если страна пока не определена, показывается нейтральный глобус.
- Расширенная диагностика теперь включает код/название страны профиля.
