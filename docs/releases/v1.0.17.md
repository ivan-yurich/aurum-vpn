# Aurum VPN Android 1.0.17

Date: 2026-05-25

## What changed

- Added Hysteria2 import support for `hy2://` and `hysteria2://` links.
- Added basic Hysteria v1 import support for `hysteria://` links.
- Added VLESS `packet_encoding: xudp` for generated VLESS profiles.
- Kept NaiveProxy on stable HTTP/2 mode with protected DNS and FakeDNS scoped to TUN traffic.
- Shortened the manual disconnect path to reduce UI hangs while native cleanup finishes.
- Updated QR/import hints, FAQ text, diagnostics redaction, and regression tests.

## Verification

- `flutter test`
- `flutter analyze`
- `flutter build apk --release --split-per-abi`
- `apksigner verify --verbose` passed for all split APKs with APK Signature Scheme v2.
