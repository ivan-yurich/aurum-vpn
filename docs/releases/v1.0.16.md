# Aurum VPN Android 1.0.16

Дата: 2026-05-25

## Что изменено

- DNS теперь ближе к логике NekoBox: домен самого прокси-сервера
  (`n8n-cloud.online`, `go-it.tech` и т.д.) решается локальным DNS, а DNS
  пользовательского трафика остаётся защищённым и идёт через туннель.
- FakeDNS оставлен только для TUN-входа, чтобы mixed proxy, проверки
  подключения и служебные запросы не получали лишние FakeIP-записи.
- Сохранены оптимизации 1.0.15: MTU 9000, endpoint-independent NAT,
  NaiveProxy HTTP/2 (`quic: false`) и логи только по запросу.
- Из release APK убраны debug symbols native-библиотек. Размер APK стал почти
  в два раза меньше.

## APK

- `AurumVPN-android-arm64-v8a-release.apk` - 33.98 MB

## Примечание

Перед сравнением скорости с NekoBox лучше остановить VPN, открыть Aurum заново
и подключиться к профилю заново, чтобы Android не держал старый VPN-процесс и
старый DNS cache.
