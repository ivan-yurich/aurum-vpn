# Aurum VPN Android v1.0.24

## Changes

- Added a subscription countdown to the profile/network card.
- Profiles imported from subscription links now keep an expiration timestamp
  when the subscription server provides it.
- Supports common `subscription-userinfo` metadata with `expire=...`.

## Validation

- `flutter analyze`
- `flutter test`
- `flutter build apk --release --split-per-abi`
- `apksigner verify`
