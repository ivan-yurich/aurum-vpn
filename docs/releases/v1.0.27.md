# Aurum VPN Android v1.0.27

## Changes

- Fixed the in-app updater handoff when the APK downloads but Android refuses
  to open the installer.
- The native installer bridge now copies the downloaded APK into Aurum VPN's
  cache, creates a `content://` URI through `FileProvider`, opens it with the
  APK MIME type, and grants temporary read access to matching Android package
  installers.
- Keeps the v1.0.26 visible GitHub Releases update-channel indicator.

## Validation

- `dart format`
- `flutter analyze`
- `flutter test`
- `flutter build apk --release --split-per-abi`
- `apksigner verify`
