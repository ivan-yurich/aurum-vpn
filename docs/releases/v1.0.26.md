# Aurum VPN Android v1.0.26

## Changes

- Added a visible "GitHub Releases" update channel line in the in-app Updates
  panel.
- Prepared this release specifically for validating phone-side auto-update.
- Includes the v1.0.25 stability and security hardening work:
  - throttled traffic UI updates;
  - native VPN call timeouts;
  - safer Android service cleanup;
  - guarded libbox initialization;
  - disabled Android backup/data extraction and cleartext app traffic.

## Validation

- `dart format`
- `flutter analyze`
- `flutter test`
- `flutter build apk --release --split-per-abi`
- `apksigner verify`
