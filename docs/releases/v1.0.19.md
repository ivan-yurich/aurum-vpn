# Aurum VPN Android 1.0.19

Дата: 2026-05-25

## Что изменилось

- Откатили экспериментальные Android-поля из 1.0.18, из-за которых VPN мог
  сразу останавливаться на старте.
- Вернули стабильный порядок NaiveProxy: сначала native Naive, затем HTTPS
  CONNECT fallback.
- Убрали стартовую проверку скорости туннеля, чтобы рабочий профиль не
  отклонялся до прогрева соединения.
- Оставили поддержку Hysteria/Hysteria2, VLESS Reality и DNS через туннель.

## Что тестировать

1. Импорт `naive+https://...`.
2. Подключение NaiveProxy на Wi-Fi.
3. Подключение NaiveProxy на LTE/мобильной сети.
4. Переключение NaiveProxy -> VLESS Reality -> NaiveProxy.
5. Ручная остановка VPN: интерфейс не должен зависать.
