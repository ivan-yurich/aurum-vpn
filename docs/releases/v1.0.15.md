# Aurum VPN Android 1.0.15

Дата: 2026-05-25

## Что изменено

- Поднял TUN MTU до 9000, как рекомендует документация NekoBox для скорости и
  меньшей нагрузки на CPU.
- Включил endpoint-independent NAT для TUN.
- FakeDNS теперь применяется только к TUN-входу, чтобы mixed proxy и проверки
  подключения не получали лишний FakeIP-слой.
- Для сгенерированных NaiveProxy профилей явно ставится `quic: false`, чтобы
  Naive работал через HTTP/2 и не уходил в QUIC/H3 без отдельного raw config.
- Логи sing-box больше не стримятся постоянно во Flutter во время обычной
  работы. Поток логов включается при раскрытии панели логов или подготовке
  отчёта.
- Окно добавления профиля осталось по центру экрана.
