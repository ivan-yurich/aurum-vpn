# Yurich Connect Android v1.0.37

Дата: 2026-06-08

## Что изменилось

- Исправлена проверка пинга для Hysteria/Hysteria2.
- UDP/QUIC-профили больше не помечаются `offline` из-за TCP-проверки порта.
- Для Hysteria/Hysteria2 приложение проверяет доступность endpoint через DNS и
  показывает `DNS ... ms` или `UDP ok`.
